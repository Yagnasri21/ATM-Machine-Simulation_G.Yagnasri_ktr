## Features
- **Secure Authentication**: Requires a Card Number and PIN to access account features.
- **Balance Inquiry**: View the current balance in real-time.
- **Cash Deposit**: Add funds to the account with instant balance updates.
- **Cash Withdrawal**: Securely withdraw funds with automatic balance validation (prevents overdrawing).
- **Transaction History**: Tracks and displays a detailed log of all deposits and withdrawals performed during the session.
- **OOP Architecture**: Built using Object-Oriented Programming principles with specialized classes for `User`, `Transaction`, and `ATM`.

## How to Run
1. Ensure you have Python installed on your system.
2. Open a terminal or command prompt.
3. Navigate to the project directory.
4. Run the following command:
   ```bash
   python main.py
   ```
